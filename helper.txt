--- First Important Web Dev Concept ---

Webpages are basically:

boxes inside boxes

Everything is a rectangle/div/container.

div

This is the most important HTML tag after maybe body.

A div is basically:

a container/box

Think of it like:

a card
a section
a box holding things

Most websites are just:

div inside div inside div 

div is there to make a cardboard box on some propeties so if we need to hide them, realign them etc we dont have to
individually pick them out.

<CSS>

here we have selector and then properties

if we need to change anything in body then 
 body {
    background-color: #f0f0f0;
 }

if we need to change anything in h1 heading
h1 {
    text-align: center;
    text-size: 50px;
}

Now there are different selectors like

1. Class selector: (can be called multiple times)
<p class="cuteText">Hello</p> --> HTML
.cuteText {
    color: red; --> CSS
} 

2. ID selector: (unique)
<p id="cuteText">Hello</p> --> HTML
#cuteText {
    color: Blue --> CSS
}

3. Universal Selector:
* {
    margin: 0;
}

4. multiple selectors:
p, h1 {
    text-align: center;
}

font-family: 'Gill Sans', 'Gill Sans MT', Calibri, 'Trebuchet MS', sans-serif; // Try Gill Sans
If unavailable → Gill Sans MT
If unavailable → Calibri
If unavailable → Trebuchet
Else any sans-serif font

now instead of padding we can use:
    display: flex;
    flex-direction: column;
    justify-content: space-between;

where we turn div into a flex container and then we can use flexbox properties. div becomes parent and h2 and p becomes its 
children. flex-direction will make them stack vertically. justify-content will make them space evenly.
